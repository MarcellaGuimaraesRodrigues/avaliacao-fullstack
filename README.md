# Avaliação Fullstack

Projeto em React com uma landing page simples e um fluxo de integração de dados.

## O que foi feito

- Interface em React (Vite)
- Simulação de fluxo:
  - banco de dados → filtro → transformação → validação → API → log
- Documentação do fluxo em:
  - integration-flow/flow-documentation.md
- Diagrama em:
  - integration-flow/flow-diagram.drawio
- Query SQL em:
  - integration-flow/queries.sql

## Como rodar

```bash
npm install
npm run dev
